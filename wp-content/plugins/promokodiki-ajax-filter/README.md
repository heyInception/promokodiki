# Promokodiki AJAX Filter

Контекстный AJAX-фильтр промокодов для главной страницы, рубрик промокодов и страниц магазинов.

## Установка и настройка

1. Активируйте плагин **Promokodiki AJAX Filter**.
2. Откройте **Промокоды → AJAX-фильтр**.
3. Настройте количество карточек, размер порции «Показать ещё», период популярности, сортировки, показ истёкших предложений и подписи интерфейса.
4. Проверьте главную, рубрику и страницу магазина. После приёмки вручную отключите Filter Everything и Filter Everything Pro.

При отключённом JavaScript форма работает обычным GET-запросом с перезагрузкой страницы. При включённом JavaScript выдача, адресная строка и история браузера обновляются без перезагрузки.

## Параметры URL

- `paf_category` — ID одной категории промокодов.
- `paf_brand` — ID одного бренда.
- `paf_sort` — `newest`, `popular`, `expiring` или `oldest`.
- `paf_popular=1` — эксклюзивный режим популярности за настроенный период; категория, бренд и сортировка очищаются.
- `paf_page` — номер загружаемой порции.

Недопустимые или не относящиеся к текущему контексту значения отбрасываются на сервере.

## Статистика популярности

Каждый клик по кнопке промокода учитывается, включая повторные клики одного посетителя. Дневные значения агрегируются в таблице `wp_promokodiki_click_stats`, а общий счётчик сохраняется в метаполе `_promocode_used_count`.

По умолчанию «Популярное за неделю» использует скользящее окно 7 дней. После первого включения действует холодный старт: в подборке нет товаров, пока не появятся новые клики за выбранный период.

## Откат

Для ручного отката активируйте прежние плагины Filter Everything, затем отключите Promokodiki AJAX Filter. В шаблонах сохранён базовый серверный вывод на случай, если новый плагин неактивен; перед откатом проверьте отображение на тестовом окружении.
